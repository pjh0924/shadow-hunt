# 데모 영상 스크립트 (60초)

## 장면 1 — 인트로 (0:00 ~ 0:08)
- 화면: 앱 런치 → "신호를 스캔하는 중…" 로딩
- 나레이션: "Shadow Hunt. 당신 동네 어디엔가 형상이 숨어있습니다."

## 장면 2 — 지도 (0:08 ~ 0:20)
- 화면: 지도 표시, 3개 귀신 마커 + 안개 동심원 + 사용자 위치 정확도 원
- 조작: 앱바 타이틀 600ms 롱프레스 → Mock 모드 진입 (MOCK 배지)
- 나레이션: "스팟으로 이동하세요. 헌트존(25m) 안이 돼야 카메라를 켤 수 있습니다."

## 장면 3 — 헌트존 진입 (0:20 ~ 0:30)
- 조작: D-Pad 가운데 ✦ 탭 → 가장 가까운 마커로 텔레포트
- 화면 변화:
  - 방향 인디케이터 "0m, 미지의 형상 #N"
  - 노란 경고 "⚠ 미지의 형상 #N 감지됨"
  - CTA 활성 "📷 카메라 모드 진입" (네온 글로우)
  - 시스템 알림 배너 (상단)
  - 햅틱 피드백 (medium)

## 장면 4 — 카메라 모드 (0:30 ~ 0:40)
- 조작: "카메라 모드 진입" 탭
- 화면: 라이브 카메라 + HUD
  - 좌상단 X · 중앙 REC 🔴 · TARGET 라벨 · 우상단 EMF 미터
  - 노이즈 필터 (스캔라인 + 글리치 + 비네팅)
  - 하단 CLEAR/WARN/DANGER 상태 + 큰 네온 셔터

## 장면 5 — AI 검출 (0:40 ~ 0:50)
- 조작: (에뮬) 🐞 탭 — 실기에선 사람 얼굴 비춰보기
- 화면:
  - 박스 위에 **빨간 4코너 L 브래킷** + "ENTITY-001" 라벨 + 스캔라인
  - EMF 미터 4.6+ 로 점프 (LED 5칸 풀)
  - 상태 "DANGER" 펄스
  - "⚠ ENTITY DETECTED" chip + 첫 발견 적색 비네팅 펄스
  - 햅틱 heavy-heavy-medium

## 장면 6 — 캡처 + 도감 (0:50 ~ 1:00)
- 조작: 셔터 버튼 탭 (흰 플래시)
- 자동: 사진에 박스 합성 → Preferences 저장 → 지도 복귀, 해당 마커 제거
- 조작: 우상단 📓 → 도감 그리드
- 화면:
  - 썸네일 그리드 (EMF 배지, 라벨, 날짜)
  - 탭 시 디테일: 큰 사진 + EMF/검출수/시각 + **미니맵** (캡처 좌표)
  - 공유 ↗ / 삭제 🗑

## 캡처 팁

- 에뮬레이터: `adb emu geo fix <lng> <lat> <alt> <accuracy>` 로 위치 주입
- 권한 자동 grant: `adb shell pm grant com.shadowhunt.app android.permission.CAMERA` (LOCATION/NOTIFICATIONS 도)
- 녹화: `adb shell screenrecord /sdcard/demo.mp4` → 중단 후 `adb pull`
- Pixel 7 에뮬 해상도 1080×2400 → 세로 모드, 호러 톤 적합
